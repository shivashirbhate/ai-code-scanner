# Changelog

All notable changes to the "AI Code Scanner" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-12-XX

### Added
- Initial release of AI Code Scanner extension
- Code analysis with static analysis and code smell detection
- Security vulnerability scanning
- Local LLM integration for code explanations and execution tracing
- Documentation generation from scan results
- Context menu integration for easy access
- Output panel for detailed results
- Configuration options for LLM endpoint and model

### Features
- **AI Code Scanner: Scan Current File** - Comprehensive analysis with AI insights
- **AI Code Scanner: Analyze for Code Smells** - Static analysis only
- **AI Code Scanner: Security Scan** - Security vulnerability detection
- **AI Code Scanner: Generate Documentation** - Create project documentation

### Technical
- Built with TypeScript and webpack bundling
- VS Code extension API integration
- Support for multiple programming languages
- Privacy-first design with local LLM support
